标题：QinQ

文本：基本qinq：不关心用户发来的数据是什么，只关系给不同公司发来的数据打上不同的TAG

Q dot1q //应该基于二层，不含三层结构

配置：

port link-type dot1q-tunnel

port default vlan-id

灵活QinQ

端口安全

port-security enable 开启端口安全功能

开启端口安全后，这个接口所学到的动态MAC地址就会转换成安全MAC地址

修改接口最大安全MAC地址数量

port -security max-mac-num 2

port-security aging-time min type absolute/inactivity//绝对老化，相对老化

mac地址绑定

port-security mac-address + mac-id 接口的MAC地址静态绑定，不会随着交换机的重启而丢失，写在配置文件里

解决方案

port-security mac-address sticky 基于粘滞的安全静态MAC地址表项

display mac-address security 查看安全地址表
