(base) 192:

不知什么时候，我发现terminal中的提示符变为了(base) 192:

经过查阅资料后发现 Bash 窗口前看到 (base) 通常表示正在使用 Anaconda 或 Miniconda 的环境管理工具，并且当前激活了一个名为 base 的 Conda 环境。

Conda 是一个开源的包管理系统和环境管理系统，主要用于管理 Python 包和依赖项。默认情况下，安装 Anaconda 或 Miniconda 后，base 环境会自动激活。这个环境包含了 Anaconda 或 Miniconda 的基本工具和库。

如果不想在启动终端时自动激活 base 环境，可以通过以下命令禁用自动激活：

这样，(base) 就不会在你打开新的终端窗口时自动出现了。你仍然可以通过 conda activate base 手动激活它。
